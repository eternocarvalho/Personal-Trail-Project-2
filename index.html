Elementos Básicos

Inicialmente vamos ver os elementos básicos de um arquivo HTML. Apesar de não serem visualizados na página, são essenciais para o correto funcionamento do site. Veja alguns exemplos.

  • <!DOCTYPE html>
  • <html>
  • <head>
  • <meta>
  • <title>
  • <body>

Podemos ver a seguir a funcionalidade e importância de cada um deles.

<html>

A Tag html envolve todo o corpo do nosso site. Ele é o elemento raiz do nosso documento.

<head>

Aqui incluímos informações gerais sobre o documento (metadados, título).

<meta>

De acordo com a definição retirada do site Mozila Developer Network:

O elemento HTML meta define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.

<title>

É através dessa Tag que inserimos o título do site, que fica na aba superior do browser.

<body>

Essa Tag é utilizada para incluir o conteúdo da página.
